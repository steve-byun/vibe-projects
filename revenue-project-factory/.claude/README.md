# 🤖 Claude AI 가이드 폴더

이 폴더는 Claude Code가 자동으로 프로젝트를 생성하고 관리하기 위한 참조 문서입니다.

---

## 📚 파일 목록

### 1. [MASTER_GUIDE.md](./MASTER_GUIDE.md)
**목적**: Claude가 모든 작업을 수행할 때 참고하는 마스터 가이드

**포함 내용**:
- 전체 워크플로우
- 템플릿 구조
- AdSense 통합 방법
- 배포 프로세스
- 문제 해결

**사용 시기**: 모든 요청 처리 시

---

### 2. [TEMPLATE_GUIDE.md](./TEMPLATE_GUIDE.md)
**목적**: 새로운 템플릿을 만들 때 참고

**포함 내용**:
- 기존 템플릿 설명
- 새 템플릿 만드는 방법
- 템플릿 카테고리
- 완전한 예시 코드

**사용 시기**: 새 템플릿 생성 요청 시

---

### 3. [HOW_TO_USE.md](./HOW_TO_USE.md)
**목적**: 사용자가 어떻게 요청해야 하는지 설명

**포함 내용**:
- 요청 패턴
- 요청 템플릿
- 작업 흐름
- 팁

**사용 시기**: 사용자 가이드 필요 시

---

## 🎯 사용 방법

### Claude가 이 가이드를 사용하는 방법

1. **사용자 요청 받음**
   ```
   사용자: "BMI 계산기 만들어줘"
   ```

2. **MASTER_GUIDE.md 참조**
   - 워크플로우 확인
   - 적절한 템플릿 선택
   - AdSense 코드 체크

3. **TEMPLATE_GUIDE.md 참조** (필요시)
   - 새 템플릿 필요 시
   - 유사 템플릿 찾기

4. **작업 실행**
   - 프로젝트 생성
   - 기능 구현
   - 배포 안내

5. **결과 전달**

## 💡 사용자를 위한 가이드

사용자가 Claude에게 요청할 때는 [HOW_TO_USE.md](./HOW_TO_USE.md)를 참고하세요!

**간단한 요청 예시**:
```
"BMI 계산기 만들어줘"
"텍스트 도구를 참고해서 이미지 변환기 만들어줘"
"지금 만든 프로젝트 배포해줘"
```
