### A. 마이크로 SaaS / Vertical SaaS

특정 니치 문제를 해결하는 소규모 구독형 소프트웨어. 2026년 가장 주목받는 1인 개발자 수익 모델.

| 항목 | 내용 |
|------|------|
| **구체적 예시** | Senja.io (후기 수집 도구, $1M ARR 달성), Cleanvoice (팟캐스트 AI 편집), Plausible Analytics (프라이버시 중심 웹 분석, $14.2K/월 오픈소스) |
| **성공 사례** | 한 개발자가 실패한 앱에서 30개 앱 포트폴리오로 전환 → $22K/월 달성 (1년 미만). 또 다른 사례: $0 → $62K MRR을 3개월 만에 달성 |
| **현실적 월 수익** | $1K~$20K MRR (니치에 따라 $50K+ 가능) |
| **시작 난이도** | ★★★☆☆ (3/5) — 기술력은 충분, 니치 발굴과 고객 검증이 핵심 |
| **수익까지 시간** | 3~6개월 (빠른 검증 시) / 6~12개월 (일반적) |
| **AI 활용도** | ★★★★★ — Claude로 코드 생성, 고객 인터뷰 분석, 랜딩페이지 카피, 마케팅 자동화 가능. 개발 속도 5~10배 향상 |

**2026년 트렌드**: Vertical SaaS(특정 산업 전용 도구)가 수평적 SaaS 대비 성장률 31% vs 28%로 우위. "조경업체용 CRM", "요가 스튜디오 예약 시스템" 같은 초니치가 유망. AI 에이전트 내장 SaaS가 핫 키워드. 부트스트랩 마이크로 SaaS의 이익률은 70%+.

### B. Chrome 확장 / 브라우저 도구

Chrome Web Store를 통한 브라우저 확장 프로그램 판매. AI 통합 확장이 프리미엄 가격을 정당화.

| 항목 | 내용 |
|------|------|
| **구체적 예시** | GMass (이메일 마케팅, $130K/월), Closet Tools (Poshmark 자동화, $42K/월), CSS Scan (CSS 검사, $100K+ 누적) |
| **성공 사례** | 한 개발자가 Chrome 확장으로 $4,012를 7일 만에 달성. AI 확장 시장 2023년 $1.5B → 2031년 $7.8B 전망 |
| **현실적 월 수익** | $0~$10K/월 (대부분 $0, 성공 시 $5K~$42K+) |
| **시작 난이도** | ★★☆☆☆ (2/5) — 이미 Chrome 확장 개발 경험 보유 (AI 크로스체커) |
| **수익까지 시간** | 1~3개월 (유료 모델 적용 시) |
| **AI 활용도** | ★★★★★ — AI 기능 자체가 핵심 차별점. Claude API 연동으로 "AI-powered" 확장 제작 가능 |

**2026년 트렌드**: 프리미엄 모델($4.99~$20/월 구독)이 주류. AI 기능 탑재 확장이 더 높은 구독료 정당화. 생산성, SEO, 콘텐츠 생성 분야 확장 수요 급증. 기존 AI 크로스체커 경험을 바로 활용 가능.

### C. 디지털 제품 (템플릿, 강의, eBook, 에셋)

한 번 만들어 반복 판매하는 디지털 상품. 2026년 글로벌 디지털 제품 시장 $124B+.

| 항목 | 내용 |
|------|------|
| **구체적 예시** | Notion 템플릿 ($8.7K/27일, Gumroad), Thomas Frank 번들 ($49~$179), AI 활용 가이드/eBook, 코딩 강의(Udemy/자체) |
| **성공 사례** | Notion 앰배서더 한 명이 250,000+ 템플릿 판매 → $275K+ 수입. AI 생성 Notion 템플릿으로 27일 만에 $8,731 달성 |
| **현실적 월 수익** | $500~$5K/월 (꾸준히 성장, 히트 시 $10K+) |
| **시작 난이도** | ★★☆☆☆ (2/5) — 제작은 쉬움, 마케팅이 핵심 |
| **수익까지 시간** | 1~4주 (첫 판매) / 3~6개월 (안정적 수입) |
| **AI 활용도** | ★★★★★ — Claude로 템플릿 구조 설계, eBook 집필, 강의 커리큘럼 작성 전부 가능. 생산 속도 극대화 |

**2026년 트렌드**: "Done-For-You(DFY)" 제품이 전통적 강의를 압도. 40개 비디오 강의보다 바로 복제 가능한 Notion 템플릿이 더 잘 팔림. AI 스킬/자동화 가이드가 가장 빠르게 성장하는 eBook 카테고리. 니치 특화 템플릿이 관건.

### D. 콘텐츠 — YouTube (자동화 / 페이스리스 채널)

AI 도구를 활용한 얼굴 없는(Faceless) YouTube 채널 운영.

| 항목 | 내용 |
|------|------|
| **구체적 예시** | 다큐멘터리/과학/법률 채널 (월 $43K~$250K), Shorts 채널 |
| **성공 사례** | 3개 페이스리스 채널 운영자 → 합산 $250K/월. 과학 채널 단독 $43K/월 |
| **현실적 월 수익** | $0~$2K/월 (첫 6~12개월) → $5K~$50K/월 (성공 시) |
| **시작 난이도** | ★★★☆☆ (3/5) — 콘텐츠 기획력 필요, AI 도구로 제작은 용이 |
| **수익까지 시간** | 6~12개월 (수익화 요건: 구독자 1,000명 + 시청 4,000시간) |
| **AI 활용도** | ★★★★★ — 스크립트(Claude), 음성(ElevenLabs), 편집(Pictory), 섬네일 생성 전 과정 AI화 가능. Shorts Factory 프로젝트와 직접 연결 |

**2026년 트렌드**: Shorts 수익화 + 제휴 마케팅 + 스폰서십으로 수익 다각화. YouTube의 "재활용 콘텐츠" 감시 강화 → 독창적 각도 필수. 기존 Shorts Factory 프로젝트를 활용하면 빠른 진입 가능.

| 항목 | 내용 |
|------|------|
| **구체적 예시** | Substack 유료 뉴스레터, 기술 블로그 + 제휴 마케팅, AI 트렌드 뉴스레터 |
| **성공 사례** | Substack 상위 50명 이상 작가 → 각 $1M+/년. 한 크리에이터 $100K 연매출 + 15,000 구독자 달성. 제휴 마케팅 경험 10년+ → 월 $44K+ |
| **현실적 월 수익** | 뉴스레터: $0~$2K/월 (초기) → $5K~$20K (성장). 블로그 제휴: $50~$200 (3~6개월) → $1K~$10K (1~3년) |
| **시작 난이도** | ★★☆☆☆ (2/5) — 시작은 쉬움. 꾸준함이 핵심 |
| **수익까지 시간** | 뉴스레터: 3~6개월. 블로그 SEO: 6~12개월+ |
| **AI 활용도** | ★★★★☆ — 콘텐츠 초안 작성, SEO 키워드 분석, 제휴 상품 리서치에 Claude 활용 가능. 단, 독자 신뢰를 위해 개인 관점/경험이 필수 |

**2026년 트렌드**: "뉴스레터 + 디지털 제품" 조합이 2026 최적 비즈니스 모델로 평가됨. Substack 작가 총 수입 $450M/년+. 제휴 마케팅의 78%가 SEO 기반.

| 항목 | 내용 |
|------|------|
| **구체적 예시** | AI 챗봇 구축 서비스, 문서 자동화 도구, 산업별 AI 에이전트 (의료/법률/부동산) |
| **성공 사례** | AI 자동화 에이전시 → 런칭 11개월 만에 $12K MRR. 70%+ 마진 달성. AI 에이전트 시장 2025년 $7.6B → 2030년 $50B+ 전망 |
| **현실적 월 수익** | 래퍼 SaaS: $0~$5K (경쟁 극심). 에이전시/컨설팅: $5K~$15K/월 |
| **시작 난이도** | ★★★☆☆ (3/5) — 래퍼 개발은 쉬우나 차별화 어려움. 에이전시는 영업력 필요 |
| **수익까지 시간** | 래퍼: 1~3개월. 에이전시: 1~3개월 (첫 클라이언트) |

### 1위: AI-Powered Chrome 확장 (프리미엄/구독형)

**왜 이것인가?**
- 이미 AI 크로스체커 확장을 만들어 본 실전 경험 보유
- Chrome 확장 개발의 전체 사이클(Manifest V3, 백그라운드, 콘텐츠 스크립트)을 이해
- AI 통합 확장은 일반 확장 대비 높은 구독료 정당화 가능 ($9.99~$19.99/월)
- AI 확장 시장 2031년 $7.8B 전망 — 아직 초기 진입 기회
- 개발 → 출시까지 2~4주면 충분

**현실적 목표**: 6개월 내 $3K~$10K MRR

**첫 번째 행동**: Chrome 확장 아이디어 3~5개를 브레인스토밍하고, 각각의 키워드를 Chrome Web Store에서 검색하여 경쟁 강도와 리뷰(불만)를 분석한다. 특히 "AI + 생산성", "AI + SEO", "AI + 글쓰기" 카테고리에서 기존 확장의 1-star 리뷰를 읽어 해결되지 않은 문제를 찾는다.

### 2위: Vertical 마이크로 SaaS (니치 산업 특화)

**왜 이것인가?**

- Vertical SaaS가 수평 SaaS 대비 성장률 우위 (31% vs 28%)
- 부트스트랩 시 이익률 70%+
- 1인 운영 SaaS 창업자가 전체의 39% — 혼자 해도 되는 시장

**현실적 목표**: 6~12개월 내 $5K~$15K MRR

**첫 번째 행동**: 자신이 관심 있거나 지인이 종사하는 산업 3~5개를 선정하고, 해당 산업 종사자에게 "가장 반복적이고 짜증나는 업무"를 묻는다. Reddit, 네이버 카페, 산업별 커뮤니티에서 불만 포스트를 수집한다. "조경업체", "요가/필라테스", "소규모 병원", "학원" 등 한국 시장의 비효율도 기회.

**현실적 목표**: 3개월 내 $1K~$3K/월

**첫 번째 행동**: "Claude와 함께 Chrome 확장 만들기" 또는 "AI 도구 활용 1인 개발 가이드" 같은 DFY 템플릿/가이드를 하나 만든다. Gumroad에 등록하고, 트위터/레딧 인디해커 커뮤니티에서 론칭 포스트를 작성한다.
