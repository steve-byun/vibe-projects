# 🚀 Claude에게 요청하는 방법

이 가이드는 사용자가 Claude에게 어떻게 요청하면 되는지 설명합니다.

---

## 💬 기본 요청 패턴

### 1. 새 프로젝트 만들기

**요청 예시**:
```
BMI 계산기 만들어줘
환율 변환기 만들어줘
QR 코드 생성기 만들어줘
```

**Claude가 할 일**:
1. 적절한 템플릿 선택
2. 프로젝트 생성
3. 기능 구현
4. 로컬 테스트 안내
5. 배포 방법 안내

---

### 2. 기존 프로젝트 참고해서 만들기

**요청 예시**:
```
텍스트 도구를 참고해서 이미지 변환기 만들어줘
계산기를 참고해서 팁 계산기 만들어줘
```

**Claude가 할 일**:
1. 참고 프로젝트 확인
2. 유사한 구조로 새 프로젝트 생성
3. 차이점 구현
4. 배포 안내

---

### 3. 배포하기

**요청 예시**:
```
지금 만든 프로젝트 배포해줘
Netlify에 올려줘
```

**Claude가 할 일**:
1. AdSense 코드 확인
2. Netlify 수동 배포 안내
3. 배포 후 확인 방법 안내

---

### 4. 프로젝트 수정하기

**요청 예시**:
```
계산기 색상을 파란색으로 바꿔줘
텍스트 도구에 단어 카운터 추가해줘
```

**Claude가 할 일**:
1. 해당 파일 찾기
2. 수정
3. 재배포 안내 (필요시)

---

## 🎯 효과적인 요청 방법

### ✅ 좋은 요청

```
"BMI 계산기 만들어줘. 키와 몸무게 입력하면 BMI 계산되고,
결과에 따라 저체중/정상/과체중 표시해줘"
```

**이유**: 구체적이고 명확함

### ❌ 애매한 요청

```
"뭐 좋은 거 만들어줘"
```

**문제**: 무엇을 만들지 불명확

---

## 📝 요청 템플릿

### 템플릿 1: 계산기 종류

```
[XXX] 계산기 만들어줘

예시:
- 대출 계산기
- BMI 계산기
- 팁 계산기
- 날짜 계산기
```

### 템플릿 2: 변환기 종류

```
[XXX] 변환기 만들어줘

예시:
- 단위 변환기 (길이, 무게, 온도)
- 통화 변환기
- 색상 코드 변환기
```

### 템플릿 3: 생성기 종류

```
[XXX] 생성기 만들어줘

예시:
- 비밀번호 생성기
- QR 코드 생성기
- Lorem Ipsum 생성기
```

### 템플릿 4: 게임

```
[XXX] 게임 만들어줘

예시:
- 2048 게임
- 뱀게임
- 타이핑 게임
```

---

## 🔄 작업 흐름

```
1. 요청 → Claude가 프로젝트 생성
           ↓
2. 로컬 테스트 (npx serve .)
           ↓
3. 수정 요청 (필요시)
           ↓
4. 배포 요청 → Netlify 업로드
           ↓
5. URL 확인 → 완료!
```

---

## 💡 팁

1. **구체적으로 요청하세요**
   - 기능, 색상, 레이아웃 등

2. **한 번에 하나씩**
   - 여러 프로젝트를 동시에 요청하면 복잡해집니다

3. **배포는 수동으로**
   - Netlify 웹사이트에서 드래그 앤 드롭이 가장 확실합니다

4. **AdSense 코드 확인**
   - 배포 후 페이지 소스에서 확인하세요
